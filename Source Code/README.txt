First open the Source Code Folder, then open
the Root.properties, you will see: 
db.password=YOUR_ROOT_PASSWORD
Change it to your password, and save the file.

On the file location, click on the path above.
Type cmd, -Press Enter
A Command Prompt should open.


Type the commands:
javac -cp .;mysql-connector-j-9.6.0.jar *.java
-Press Enter.

java -cp .;mysql-connector-j-9.6.0.jar Project3App
-Press Enter.


Repeat this process:
On the file location, click on the path above.
Type cmd, a new Command Prompt should open.

Type the commands:
javac -cp .;mysql-connector-j-9.6.0.jar *.java
-Press Enter.

java -cp .;mysql-connector-j-9.6.0.jar AccountantApp
-Press Enter.
